HTML 5 Y CSS 3

HTML 5

28 de octubre de 2014 se aprobó la Recomendacion HTML 5, culminando asi diez alis de trabajo. En 2004 se formó el WHATWG, un grupo informal de individuos y empresas decididos a impulsar el desarrollo del HTML, que se encontraba bloqueado en el W3C desde el año 2000. En 2007 el W3C retornó el trabajo sobre el HTML y siete años desués alcanzó el objetivo.

Preo el HTML 5 no era ,as de que el primer paso. El 1 de noviembre de 2016 se aprobó la recomendación HTML 5.1, amppliando t clarificando el HTML 5, t el W3C ya está trabajando en el HTML 5.2, previsto para finales de 2017.

CSS 3

Desde la aprobación de CCS 2 en 1998, el W3C ha estado trabajando en la siguiene generacion de las hojas de estilo, las CSS 3. Las Recomendaciones CSS 3 se planificaron como un conjunto de recomendaciones en vez de como un decumento unico. El proceso ha resultado largo y poco productivo: las primeras recomendacioes CSS 3 no se aprobarion hasta 2011

Aunque los navegadores admiten muchas caracteristicas de CSS 3, la publicacion de HTML 5 no ha aceerado el desarrollo de CSS 3.
